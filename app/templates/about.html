Project

About This System

Decision support platform for crop recommendation using machine learning and explainability.

Problem Statement

Farmers often choose crops through intuition or local habit. This system adds a data-assisted recommendation layer based on measurable soil and climate conditions.

Core Inputs

  • Nitrogen, Phosphorus, Potassium
  • Temperature, Humidity, Soil pH, Rainfall

Tech Stack

  • Backend: Flask
  • ML: scikit-learn, XGBoost, LightGBM, CatBoost
  • Explainability: SHAP
  • Deployment: Gunicorn + Render

REST API

POST /predict with JSON payload:

{
  "N": 90,
  "P": 42,
  "K": 43,
  "temperature": 21,
  "humidity": 82,
  "ph": 6.5,
  "rainfall": 203
